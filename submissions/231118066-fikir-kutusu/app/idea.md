# Track 3: Idea Librarian (Fikir Kütüphanecisi)

**Geliştirici:** Alperen ERİ (231118066)

## Proje Tanımı

"Fikir Kütüphanecisi", kullanıcıların farklı platformlardan (WhatsApp, Not Defteri, Web vb.) topladığı dağınık, düzensiz ve tekrarlanan notları anlamlı bir yapıya kavuşturan yapay zeka destekli bir mobil uygulamadır.

## Temel Özellikler

- **AI Analizi:** Karmaşık metinleri analiz ederek ana fikirleri ayıklar.
- **Kategorizasyon:** Her notu otomatik olarak BAŞLIK | ÖZET | KATEGORİ formatına sokar.
- **Temizlik:** Metin içindeki gereksiz tekrarları ve anlamsız kalabalığı siler.
- **Görsel Kart Yapısı:** Düzenlenen fikirleri kullanıcıya profesyonel kartlar halinde sunar.

## Teknik Detaylar

- **Framework:** React Native (Expo)
- **Yapay Zeka:** Google Gemini 2.0 Flash-Exp API
- **Mimari:** REST API (Fetch) üzerinden asenkron veri işleme.

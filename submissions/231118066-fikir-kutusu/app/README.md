## 🛠️ Kurulum ve Çalıştırma

Projeyi yerel ortamınızda çalıştırmak için:

1. Depoyu klonlayın: `git clone <repo-url>`
2. Bağımlılıkları yükleyin: `npm install`
3. Uygulamayı başlatın: `npx expo start`
